Hello, World. Built the World First.

Posted on Jul 14, 2025

Programlamaya ilk başladığınızda, genelde dilin output fonksiyonunu kullanarak terminale “Hello World” yazdırırsınız. Bu, programlama dünyasına attığınız ilk adımdır;

# Python
print("Hello, World!")

// Java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

// Go
package main
import "fmt"

func main() {
    fmt.Println("Hello, World!")
}

// Rust
fn main() {
    println!("Hello, World!");
}

-- Haskell
main = putStrLn "Hello, World!"

Az önce internette buna dair bir geyik gördüm. Adam diyor ki:

“Binlerce satır kod yazdım, hiç hata almadım.” Sonraki sahnede kodu gösteriyor… Tahmin ettiğiniz gibi: Binlerce satır Hello, World yazmış.

Bu sahne beni tetikledi ve bir çılıgınlık yapma isteği uyandırdı. Henüz zamanım yok ama ilk fırsatta yapacağım. Neyi mi? Aşağıya yazıyorum:

  • Günün sonunda sadece terminale “Hello, World!” yazdırılacak.
  • Ama bunu Assembly ile yapacağım.
  • Yetmedi: O assembly kodunu çalıştıracak assembler’ı da ben yazacağım.
  • Son olarak asıl psikopatlık olan assembly’nin çalışacağı işlemciyi VHDL ya da Verilog ile ben tasarlayacağım.

Umarım bu projeye zaman bulurum da, kendi yarattığım dünyaya ilk selamı veririm :)
